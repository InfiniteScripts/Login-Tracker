Track the login times and IP address of users. Tracks City, State and Zip of your user's logins. Displayed in sortable table and exportable to CSV.

This plugin adds a column to the users table titled "Last login". This is the last time that user logged into wordpress admin. Click this column to be taken to a report of all the users logins which can be sorted and exported to csv.
